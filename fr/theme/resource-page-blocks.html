Blocs de page de ressource

OctopusViewer fournit 4 blocs différents que vous pouvez placer sur les pages de ressource.

Octopus viewer

Disponible pour: contenu, média.

C’est la visionneuse complète, avec deux barres latérales. Une contient un sélecteur de média et l’autre contient les métadonnées du média sélectionné.

Le sélecteur de média est désactivé sur la page d’un média.

Octopus viewer sélecteur de média

Disponible pour: contenu.

C’est le sélecteur de média. Il affiche les médias du contenu et permet aux utilisateurs d’en sélectionner un. Quand d’autres blocs sont présent sur la page (Octopus viewer vue du média et Octopus viewer métadonnées du média), cela les actualise.

Octopus viewer vue du média

Disponible pour: contenu, média.

C’est ici que le média est affiché.

Il peut être contrôlé par le bloc « Octopus viewer (media selector) ».

Octopus viewer métadonnées du média

Disponible pour: contenu, média.

Cela affiche les métadonnées du média sélectionné.

Il peut être contrôlé par le bloc « Octopus viewer (media selector) ».
