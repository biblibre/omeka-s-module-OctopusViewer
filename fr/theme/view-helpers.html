View helpers

octopusViewer

Vous pouvez utiliser ce view helper pour afficher la visionneuse pour tous les médias d’un contenu comme ça:

<?= $this->octopusViewer()->forItem($itemRepresentation) ?>

Ou vous pouvez afficher n’importe quels médias correspondants à certains critères:

<?= $this->octopusViewer()->viewer($media_query, $viewer_title) ?>

Par exemple,

<?= $this->octopusViewer()->viewer(
    [
        'media_type' => 'image/jpeg',
        'item_id' => $item->id(),
    ],
    'Pictures'
) ?>

La liste des paramètres disponibles se trouve dans la documentation développeur Omeka S (référence API).
